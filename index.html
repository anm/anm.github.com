HAI!

This is the homepage of Alistair MacLeod (anm), and his amateur radio station, MM0HAI.

Travel

I am cycle touring in Latin America. I'm starting in Mexico with the general idea of going to Patagonia.

I have attached a solar panel to the bike and made a charger for my laptop so I should be able to write even when away from civilisation for a while. I'm also making a keyboard for the bike handlebars so I can write with an Android phone while on boring roads. It was only half made before I left but I've got tools and hopefully the parts I will need to finish it.

In Winter 2011-2012 I went to the Canary Islands to try and hitch a lift in a boat to South America. I didn't get there but liked the Canaries anyway. Blog posts covering this adventure are tagged "serbla".

See also photos on flickr, and my twitter account for updates.

Work

Recently I have been building a technomadic bicycle. This is a bike fitted with electronic and computing gadgets to enable a lifestyle of travel while remaining connected with technology. Off-grid with solar power and online when needed. I'm now on the road but still working on it.

I occasionally take on freelance software development jobs. Previously I have worked as a software engineer at internet startups on distributed systems, machine learning, and both front and back end web development.

Code

Have a look at my javascript implementation of Conway's Game of Life. It's colourful and rather nice on the iPad where it can be installed as an offline web app. Read more about the features and implementation.

I'm starting to put code on github. I use Stack Overflow as anm.
